## 🛡️ Características de Seguridad del Sistema (Cyber-Shield)

Para mitigar intrusiones o alteraciones maliciosas en la consola de telemetría de ARGOS, el sistema incluye:
*   **Obfuscación Base64:** Las funciones lógicas que evalúan eventos de seguridad se encuentran codificadas para evitar inspecciones automatizadas por inyección de scripts externos.
*   **Filtro de Clic Derecho:** Inhabilita el menú contextual para mitigar la inspección básica de elementos HTML.
*   **Filtro de Atajos de Teclado:** Captura y bloquea las llamadas a teclas calientes de depuración (`F12`, `Ctrl+Shift+I`, `Ctrl+Shift+J`, `Ctrl+Shift+C`, `Ctrl+U`, `Ctrl+S`).

Si detectas un fallo de seguridad real o una fuga de datos en el sistema simulado:
1.  **NO abras un Issue público** en GitHub para evitar que otros usuarios lo exploten.
2.  Envía un correo privado de reporte detallado de vulnerabilidad a los mantenedores del proyecto.
3.  Evaluaremos el reporte y daremos respuesta en un lapso de 48 horas con el parche de seguridad respectivo.
